Hound Around Town

Hound Around Town is an application developed for companies allowing them to manage their personal pup clientele, along with contracted drivers to transport pets to their specific destinations.This application in particular fills the need presented by Hounds Around Town, but the application can be customized to fit any businesses unique model.

Using Sequelize and MySQL the companies information is stored in a database housing the drivers, and clients. This node app uses also incorporates SweetAlert2 for confirmation

Eat-Da-Burger

Eat-Da-Burger! is a restaurant app that lets users input the names of burgers they'd like to eat.Whenever a user submits a burger's name, your app will display the burger on the left side of the page -- waiting to be devoured.

This project allows you to create a burger logger with MySQL, Node, Express, Handlebars and a homemade ORM (yum!).

Follows the MVC design pattern; using Node and MySQL to query and route data in the app, and Handlebars to generate HTML.
